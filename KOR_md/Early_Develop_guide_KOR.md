### 목표

- 목표는 “투자 예측”이 아니라, 실제 해외주식(종목) 가격 데이터를 사용해 다양한 자산가격/주식 이론을 **직접 시뮬레이션하고 학습**하게 만드는 금융·경제 문해력 교육용 인터랙티브 웹앱을 만드는 것이다.
- 사용자는 이론을 체크박스로 선택하고, 실행 버튼을 누르면 선택한 이론별로 결과(그래프/핵심 수치)와 쉬운 설명(단계적, 매우 명확)을 동시에 확인한다.
- **가정(파라미터)을 바꾸면 결과가 바뀐다**를 체험하는 What-if(민감도) 슬라이더가 핵심이다.

---

### 중요 원칙

- CAPM 등 이론은 “단기 방향성 예측”이 아니라 “기대수익/요구수익률/위험/가정/민감도”를 산출·해석하는 방식으로 구현한다.
- 앱의 메시지: 같은 데이터라도 이론/가정이 다르면 결론이 달라진다. 사용자는 이를 조작/체험으로 학습한다.
- 제출물은 1~2분 데모 영상이므로, 화면 전환 없이도 **선택→실행→결과**가 강하게 드러나야 한다.

---

### 데이터

- 해외주식 티커 입력 또는 검색 (예: AAPL, MSFT, SPY 등).
- 최소 MVP: 샘플 데이터(JSON/CSV)를 내장해도 됨.
  - 되도록, 샘플데이터(JSON)을 사용하는 방식은 지양.
- 확장: 실제 데이터는 크롤링/외부 API로 가져올 수 있도록 구조만 열어둘 것(필수는 아님).
- 필수 데이터: 일별 종가 (가능하면 배당/분할 조정 종가), 날짜.

### 메인 사용자 흐름 (UX)

1) Landing / Setup 화면
   - 티커 입력(해외주식)
   - 기간 선택(예: 1Y/3Y/5Y/custom)
   - 이론 선택 체크박스 리스트 (복수 선택 가능)
   - 체크박스에 마우스 오버하면 1~2줄 툴팁(이론이 “무엇을 계산하는지”만 간단히)
   - “Run / 실행” 버튼

<br>

2) Results 화면(또는 같은 페이지에서 아래로 확장)
   - 선택한 이론마다 ‘카드(섹션)’가 생성됨
   - 각 이론 카드 구성(고정):
     A. 쉬운 설명(이 이론이 무엇을 계산하는가 / 어떤 가정을 하는가 / 출력의 의미)
     B. 핵심 수치 3개(이론별로 정의)
     C. 그래프 1~2개(이론별로 정의)
     D. What-if 슬라이더 1~2개(가정 변화 → 즉시 그래프/수치 업데이트)
   - 최상단에는 공통 요약 바:
     - **선택한 이론/가정이 달라지면 결론이 달라진다**를 한 줄로 보여주고,
     - 각 이론의 핵심 결과를 표 형태로 비교(가능하면).

### 구현할 투자 이론 (최소 6개, 권장 7~8개)
- 아래 이론들을 “교육용 시뮬레이션”으로 구현.
- 각 이론은 ‘예측’이 아니라 ‘모델 산출물+해석’을 보여줘야 함.
  
<br>

1) CAPM (필수)
- 산출: 베타(β) 추정, 요구수익률 E[Ri], SML(가능하면)
- 입력/가정 슬라이더: 무위험수익률 Rf, 시장위험프리미엄 (E[Rm]-Rf)
- 그래프: (a) SML 상에서 종목 점 표시 또는 (b) 기간별 β 추정 변화
- 핵심 수치 예: β, 요구수익률, 과거 실현수익률(비교용)

<br>

2) Fama-French 요인모형(가능하면 3-factor 또는 단순화)
- 산출: 요인노출 계수, 기대수익(또는 설명력)
- 슬라이더/선택: 추정 기간, 요인 버전(3-factor vs market-only)
- 그래프: 요인 기여도(막대) 또는 예측(설명) 수익률 vs 실현수익률(교육용)

<br>

3) Momentum 전략 시뮬레이터(경험적 규칙)
- 산출: 룩백 기간 기반 신호, 전략 누적수익 vs Buy&Hold, 최대낙폭
- 슬라이더: 룩백 기간(예: 3/6/12개월), 리밸런싱 주기
- 그래프: 전략 가치곡선 vs 벤치마크

<br>

4) Mean Reversion(평균회귀) 신호 시뮬레이터
- 산출: z-score/편차 기반 신호, 전략 성과 vs Buy&Hold
- 슬라이더: 이동평균 기간, 진입 임계치(z)
- 그래프: 가격과 평균선 + 신호 구간 표시, 전략 가치곡선

<br>

5) Risk Metrics 대시보드(필수)
- 산출: 변동성, 최대낙폭(MDD), 회복기간, (선택) VaR
- 슬라이더: 관측 창(rolling window)
- 그래프: 낙폭(drawdown) 차트 + 히트맵(선택)

<br>

6) Efficient Frontier / Mean-Variance(가능하면, 단순화 가능)
- 멀티자산 포트폴리오(종목+SPY+채권대용 ETF 등 프리셋 제공)
- 산출: 효율적 프론티어, 최소분산 포트폴리오, 사용자가 선택한 점
- 슬라이더: 위험회피(λ) 또는 목표 수익률
- 그래프: 리턴-리스크 공간의 프론티어

<br>

7) DCF “할인율 민감도” 미니 랩 (가격평가 관점)
- 입력: 성장률 g, 할인율 r (사용자 슬라이더)
- 출력: 이론적 가치 변화(민감도 차트)
- 단, 실제 주가와 연결은 “가정 변화 → 가치 변화” 교육용으로만.

### 디자인 초기 설정

- iOS 느낌의 둥근 카드/버튼, 반응형, 매우 깔끔한 UI
- 다크모드 기반: 블랙~남색 사이(딥 네이비) 배경
- 포인트 컬러: 노란색(강조/선택/버튼/하이라이트)
- 중요 UX: 
  - 체크박스 hover 툴팁(간단 설명)
  - 실행 후에는 동일 내용을 “자세하고 쉬운 설명 + 그래프”로 확장

### 초기 예상 산출물

1) 화면 IA(정보구조) + 컴포넌트 목록
2) 각 이론별 **설명 텍스트(초심자용)** 초안(짧고 명확하게)
3) 각 이론별 출력물 정의(핵심 수치 3개 + 그래프 1~2개 + 슬라이더 1~2개)
4) MVP 범위 확정(가장 먼저 구현할 것들)
5) (가능하면) 개발용 폴더 구조 및 주요 함수 시그니처 설계
6) 데모 영상(1~2분)용 스토리보드: 어떤 장면을 순서대로 보여줄지

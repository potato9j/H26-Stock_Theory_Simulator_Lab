1. 프로젝트 개요

Interactive Stock Theory Lab은 해외주식 데이터를 기반으로 금융 이론을 학습할 수 있도록 설계된 교육용 웹 애플리케이션입니다. 핵심 목표는 단기 가격 예측이 아니라, 같은 데이터에서도 이론과 가정에 따라 해석이 달라진다는 점을 사용자에게 체험시키는 것입니다.

2. 기획 배경 및 목적

  • 금융 이론 학습이 수식 중심으로 고립될 때 발생하는 이해 한계를 완화
  • 실제 시장 데이터와 이론을 연결해 직관적 학습 경험 제공
  • 투자 권유가 아닌, 모델 가정/민감도/한계를 비교 학습하는 교육 실험실 구현

3. 핵심 가치 제안

  • 비예측 중심: 본 서비스는 예측 엔진이 아니라 이론 적용 도구입니다.
  • 비교 학습: 이론별 결과를 한 화면에서 비교합니다.
  • 상호작용: What-if 슬라이더로 가정을 바꾸면 결과가 즉시 반영됩니다.
  • 설명 강화: 결과 수치뿐 아니라 "왜 이런 결과가 나왔는지"를 구조적으로 제공합니다.

4. 사용자 시나리오

  1. 사용자가 티커(예: AAPL, MSFT, NVDA, SPY)를 입력
  2. 기간(1M/6M/1Y/3Y/5Y/Custom) 및 이론 선택
  3. Run 실행 후 데이터 수집/정규화/계산
  4. 비교표 및 이론 카드(지표/그래프/설명/슬라이더) 확인
  5. 가정 변경 후 결과 차이를 재확인

6. 기술 아키텍처

6.1 Frontend

  • HTML + CSS + JavaScript
  • Tailwind CSS(CDN), Chart.js
  • 카드 기반 반응형 UI

6.2 Backend

  • Node.js 내장 http 서버
  • API 프록시 엔드포인트: /api/price
  • Alpha Vantage 호출, 캐시, 레이트리밋 큐 처리

7. 데이터 처리 및 계산 파이프라인

  1. 티커/기간 파라미터 수신
  2. 외부 API 호출(일/주 데이터 자동 선택)
  3. 표준 스키마 정규화: {t,o,h,l,c,v}
  4. 수익률/지표/전략/회귀 계산
  5. 비교표 및 차트 렌더링
  6. 슬라이더 입력 시 즉시 재계산/재렌더링

9. 로컬 실행 방법

  1. Node.js 설치 확인 (node -v)
  2. .env에 API 키 설정
  3. node server.js 실행
  4. 브라우저에서 http://localhost:3000 접속

10. 결과물 및 해석 방식

  • 이론별 KPI 3종 이상 제공
  • 차트 + 테이블 + 설명문 동시 제공
  • 설명 구조: 이론 가정 -> 데이터 특징 -> 결과 해석
  • 민감도 조절로 결론 변화 체험

11. 주요 이슈와 해결

  • API 제한: 무료 엔드포인트 중심 폴백 + 호출 간격 제어 + 캐시 적용
  • CORS/키 노출: 브라우저 직접 호출 대신 서버 프록시 구조로 전환
  • 반응형 차트: 컨테이너/축 설정 최적화로 오버플로우 개선
  • 설명 품질: 초보자 기준의 Why 설명 강화

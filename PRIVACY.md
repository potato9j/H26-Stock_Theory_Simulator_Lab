# Privacy Policy / 개인정보 처리방침 (국문/영문)

Last updated: 2026-01-08

## English

### 1. No Personal Data Collection
This project does not collect personal information such as names, emails, phone numbers, IDs, or payment data.
The only user input is a stock ticker symbol, which is used to fetch market data.

### 2. Technical Data (Hosting Logs)
If this project is hosted on a third-party platform, the hosting provider may collect standard server logs
(IP address, user agent, timestamps) for security and operational purposes. This project does not actively
track or store those logs.

### 3. Cookies and Analytics
This project does not use cookies, tracking pixels, or analytics tools.

### 4. Third-Party Services
Market data is retrieved from third-party providers (e.g., Alpha Vantage). Those providers may log API requests
from the server. This project does not control their data practices.

### 5. Data Retention
User inputs are not stored. Any caching is temporary and in-memory to improve performance and is cleared on restart.

### 6. Changes to This Policy
This policy may be updated without notice. Continued use of the project indicates acceptance of the updated policy.

## 한국어 (Korean)

### 1. 개인정보 수집 여부
본 프로젝트는 이름, 이메일, 전화번호, 주민등록번호, 결제 정보 등 개인정보를 수집하지 않습니다.
사용자 입력은 해외주식 티커 심볼이며, 이는 시장 데이터를 불러오기 위한 용도로만 사용됩니다.

### 2. 기술 로그 (호스팅 환경)
프로젝트가 외부 호스팅 플랫폼에 배포될 경우, 호스팅 제공자는 보안 및 운영 목적의 기본 서버 로그
(IP 주소, 브라우저 정보, 접속 시간 등)를 수집할 수 있습니다. 본 프로젝트는 이러한 로그를
직접 수집하거나 저장하지 않습니다.

### 3. 쿠키 및 분석 도구
본 프로젝트는 쿠키, 트래킹 픽셀, 웹 분석 도구를 사용하지 않습니다.

### 4. 제3자 서비스
시장 데이터는 외부 제공업체(예: Alpha Vantage)로부터 제공되며, 해당 업체는 API 요청 로그를
수집할 수 있습니다. 해당 정책은 본 프로젝트의 통제 범위를 벗어납니다.

### 5. 보관 및 파기
사용자 입력은 저장되지 않으며, 성능 개선을 위한 캐시는 메모리 기반 임시 저장이고 서버 재시작 시 삭제됩니다.

### 6. 정책 변경
본 정책은 사전 고지 없이 변경될 수 있으며, 변경 후 사용은 동의로 간주됩니다.
